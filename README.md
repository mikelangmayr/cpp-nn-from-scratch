# Neural Network from Scratch (C++)

A simple neural network implementation in modern C++17 with no external ML libraries.

## Features
- C++17
- CMake build
- Catch2 unit tests
- Feedforward network with sigmoid activation

## Build & Run
```bash
cmake -B build
cmake --build build
./build/main
```

## Run Tests
```bash
./build/tests
```
